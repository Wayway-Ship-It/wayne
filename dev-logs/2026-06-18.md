## 遇到的问题

1. **langchain4j 1.0.0 API 重命名**: `ChatLanguageModel` → `ChatModel`, `Response<AiMessage>` → `ChatResponse`。通过 `javap` 反编译确认新 API 后修复。
2. **Git Bash 中文编码**: curl 命令行直接传中文 JSON 会导致编码错误（400 Bad Request），通过 `echo -n > file` + `-d @file` 解决。
3. **DashScope 嵌入 API batch≤10**: `EmbeddingStoreIngestor` 一次性调用 `embedAll()` 超过限制。改为手动分批（每批 8 个段落）。
4. **Chroma 向量数据库**: Phase 2 使用 InMemory 存储，后续可切换 Chroma 持久化方案。
