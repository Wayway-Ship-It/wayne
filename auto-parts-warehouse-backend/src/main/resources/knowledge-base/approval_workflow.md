# 审批流程详细说明

## 审批概述

系统中的入库单和出库单都需要经过审批流程。审批功能确保出入库操作的合规性。

角色权限：
- **普通用户**：可以创建入库单、出库单，但不能审批
- **管理员**：可以审批入库单和出库单

## 入库审批流程

### 第一步：创建入库单
1. 操作员进入「入库管理」页面
2. 点击「创建入库单」
3. 选择入库类型（采购入库、退货入库等）
4. 选择供应商
5. 添加配件，填写入库数量和单价
6. 点击「提交」按钮

### 第二步：等待审批
- 入库单提交后状态变为「待审核」
- 入库单会出现在管理员的待审批列表中

### 第三步：管理员审批
1. 管理员进入「入库管理」页面
2. 找到状态为「待审核」的入库单
3. 点击「审批」按钮
4. 审核入库信息是否正确
5. 选择「通过」或「拒绝」
6. 如果拒绝，需要填写拒绝原因

### 第四步：库存更新
- 审批通过：系统自动更新库存（增加对应配件的库存数量）
- 审批拒绝：库存不变，入库单状态变为「已拒绝」

## 出库审批流程

### 第一步：申请出库
1. 操作员进入「出库管理」页面
2. 点击「申请出库」
3. 选择出库类型：
   - 销售出库：卖给客户的正常出库
   - 调拨出库：仓库之间的调拨
4. 选择客户
5. 添加配件，填写出库数量
6. 点击「提交审批」

### 第二步：库存检查
- 系统自动检查出库配件的库存是否充足
- 如果库存不足，会提示无法提交出库申请

### 第三步：管理员审批
1. 管理员进入「出库管理」页面
2. 查看待审批的出库单
3. 点击「审批」按钮
4. 核实出库信息
5. 选择「通过」或「拒绝」

### 第四步：库存扣减
- 审批通过：系统自动扣减库存
- 审批拒绝：库存不变

## 审批注意事项

- 审批操作不可撤销，请仔细核对后再操作
- 审批通过后库存立即生效
- 被拒绝的单据可以重新提交
- 建议审批前核对配件的实际库存情况
