# 库存管理详细指南

## 库存状态说明

库存管理页面展示每个配件的实时库存状态：

| 状态 | 说明 |
|------|------|
| 正常 | 当前库存 > 安全库存 |
| 预警 | 当前库存 ≤ 安全库存 |
| 缺货 | 当前库存 = 0 |

## 安全库存设置

安全库存是库存预警的阈值，当配件库存数量低于或等于安全库存时，系统会自动标记为预警状态。

### 修改安全库存

1. 进入「库存管理」页面
2. 找到需要修改的配件行
3. 点击「设置安全库存」按钮
4. 输入新的安全库存数值
5. 点击「保存」

### 推荐安全库存值

- 常用配件：安全库存不少于 10 个
- 不常用配件：安全库存不少于 3 个
- 易损件：安全库存不少于 20 个

## 库存预警机制

### 自动预警

- 系统会在每次出入库操作后自动检查库存
- 当库存 ≤ 安全库存时，配件状态变为"预警"
- 预警配件在库存列表中会有醒目标识（黄色/红色高亮）

### 预警邮件

- 系统支持一键发送预警邮件
- 在库存管理页面点击「发送预警邮件」按钮
- 系统会自动向管理员邮箱发送包含所有预警配件的清单
- 邮件内容包括：配件名称、当前库存、安全库存、差异数量

### 查看低库存配件

- 在库存管理页面，使用筛选功能
- 选择「库存不足」筛选项
- 或者在 AI 助手中输入"查看库存不足的配件"

## 库存查询

### 按配件查询

- 在搜索框中输入配件名称或编码
- 支持模糊搜索
- 查询结果显示配件名称、当前库存、安全库存、单位

### 库存列表排序

- 点击表头的「当前库存」列可排序
- 支持升序和降序排列
- 方便快速找到库存最多或最少的配件

## 库存变动记录

每次出入库操作都会自动更新库存：
- 入库审批通过：库存增加
- 出库审批通过：库存减少
- 盘点差异调整：库存更新为实际盘点数量
