# 供应商管理详细指南

## 供应商信息字段说明

| 字段 | 说明 | 是否必填 |
|------|------|----------|
| 供应商名称 | 企业全称 | 是 |
| 联系人 | 主要对接人 | 是 |
| 联系电话 | 手机或座机 | 是 |
| 联系地址 | 详细地址 | 否 |
| 备注 | 额外说明 | 否 |

## 新增供应商步骤

1. 点击左侧导航菜单「供应商管理」
2. 进入供应商列表页面，点击右上角「新增供应商」按钮
3. 在弹出的表单中填写供应商信息
4. 确保供应商名称和联系电话准确无误
5. 点击「确定」按钮保存

## 编辑供应商

1. 在供应商列表中找到需要编辑的供应商
2. 点击该行的「编辑」按钮
3. 修改需要更新的信息
4. 点击「保存」按钮

## 删除供应商

1. 在供应商列表中找到需要删除的供应商
2. 点击该行的「删除」按钮
3. 系统会弹出确认对话框
4. 确认后即可删除（注意：有关联入库单的供应商可能无法删除）

## 查询供应商

- 支持按供应商名称模糊搜索
- 支持按联系人姓名搜索
- 在搜索框中输入关键词，按回车或点击搜索按钮

## 供应商与入库关联

- 创建入库单时必须选择一个供应商
- 如果需要的供应商不在列表中，可以点击「新增供应商」快速添加
- 入库单保存后，供应商信息会关联到该入库单
