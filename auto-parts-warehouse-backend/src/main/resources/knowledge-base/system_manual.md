# 汽车配件仓库管理系统操作手册

## 一、系统概述
本系统是一个汽车配件仓库管理系统，主要功能包括：配件管理、出入库管理、库存管理、盘点管理、供应商管理、客户管理等。

## 二、功能模块说明

### 2.1 用户管理模块
- 管理员可以管理系统用户账号
- 支持添加、编辑、删除用户
- 用户分为管理员和普通用户两种角色
- 管理员具有完整权限，普通用户权限受限

### 2.2 配件信息管理模块
- 管理配件的基本信息
- 支持配件的分类管理
- 配件信息包括：配件名称、编码、规格、单位、安全库存等
- 可以通过配件名称、编码进行搜索

### 2.3 入库管理模块
- 创建采购入库单
- 选择供应商，填写入库配件信息
- 入库单需要审核确认后生效
- 支持批量入库操作

### 2.4 出库管理模块
- 创建销售出库单和调拨出库单
- 选择客户，填写出库配件信息
- 出库单需要审批通过后才能执行
- 出库时自动扣减库存

### 2.5 库存管理模块
- 实时查看库存状态
- 支持库存预警功能
- 可以筛选库存不足的配件
- 提供一键发送预警邮件功能

### 2.6 盘点功能
- 创建循环盘点单
- 选择需要盘点的配件
- 录入实际盘点数量
- 自动计算盘盈盘亏

### 2.7 供应商管理模块
- 管理供应商信息
- 支持按联系人查询供应商
- 供应商信息包括：名称、联系人、电话、地址等

### 2.8 客户管理模块
- 管理客户信息
- 支持按联系人查询客户
- 客户信息包括：名称、联系人、电话、收货地址等

## 三、常见操作指南

### 3.1 如何新增供应商
1. 进入供应商管理页面
2. 点击"新增供应商"按钮
3. 填写供应商信息：名称、联系人、电话、地址等
4. 点击"保存"按钮完成新增

### 3.2 如何创建入库单
1. 进入入库管理页面
2. 点击"创建入库单"按钮
3. 选择供应商（如果没有供应商可以新增）
4. 添加入库配件，填写数量和单价
5. 点击"提交"按钮提交审核

### 3.3 如何申请出库
1. 进入出库管理页面
2. 点击"申请出库"按钮
3. 选择出库类型（销售出库/调拨出库）
4. 选择客户（如果没有客户可以新增）
5. 添加出库配件，填写出库数量
6. 点击"提交审批"按钮

### 3.4 如何创建盘点单
1. 进入盘点管理页面
2. 点击"创建盘点单"按钮
3. 选择需要盘点的配件
4. 系统会自动填写配件分类
5. 保存盘点单后开始盘点

### 3.5 如何修改个人信息
1. 点击右上角用户名
2. 选择"个人信息"
3. 修改姓名、电话等信息
4. 点击"保存"按钮

## 四、审批流程说明

### 4.1 入库审批流程
1. 操作员创建入库单
2. 系统自动保存为待审核状态
3. 管理员审核通过后入库单生效
4. 库存自动增加

### 4.2 出库审批流程
1. 操作员创建出库单
2. 出库单进入待审批队列
3. 管理员审批通过后出库单生效
4. 库存自动扣减

## 五、库存预警机制

### 5.1 预警条件
- 当配件当前库存数量小于等于安全库存时触发预警
- 系统每天自动检查库存状态
- 可以手动触发预警邮件发送

### 5.2 预警通知方式
- 系统自动发送邮件给管理员
- 邮件内容包含：配件名称、当前库存、安全库存

## 六、智能AI助手

- 支持自然语言查询配件库存
- 支持快速跳转到功能页面
- 提供操作指引说明

- "怎么新增供应商" - 获取操作指引

## 七、注意事项

### 7.1 数据安全
- 定期备份数据库
- 不要泄露系统账号密码
- 敏感操作需要管理员权限

### 7.2 操作规范
- 出入库操作需要仔细核对数量
- 盘点时确保数据准确
- 审批操作需要认真审核

### 7.3 系统维护
- 定期清理无效数据
- 检查系统运行状态
- 及时更新系统版本
