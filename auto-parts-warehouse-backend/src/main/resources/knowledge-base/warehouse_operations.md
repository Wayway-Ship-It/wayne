# 仓库日常操作指南

## 日常操作流程

### 每日工作清单

1. **早晨检查**
   - 登录系统
   - 查看库存预警列表
   - 检查待审批单据数量
   - 查看前一天出入库记录

2. **入库操作**
   - 收到货物后核对送货单
   - 在系统中创建入库单
   - 填写准确的配件信息、数量、单价
   - 提交入库单等待审批

3. **出库操作**
   - 收到出库需求后确认库存
   - 在系统中创建出库单
   - 选择正确的出库类型和客户
   - 提交审批

4. **日常盘点**
   - 选择需要盘点的配件
   - 实际清点数量
   - 在系统中录入盘点结果
   - 处理盘盈盘亏

5. **下班前操作**
   - 确认所有单据已处理
   - 查看库存预警状态
   - 如需要发送预警邮件

## 配件管理操作

### 新增配件
1. 点击左侧菜单「配件管理」
2. 点击「新增配件」按钮
3. 填写配件信息：
   - 配件编码（唯一标识）
   - 配件名称
   - 配件分类（选择已有分类或新建）
   - 规格型号
   - 单位（个、箱、套等）
   - 单价
   - 安全库存
4. 点击「保存」按钮

### 配件分类管理
- 在配件管理页面可以管理配件分类
- 支持添加、编辑、删除分类
- 常见的汽车配件分类：
  - 发动机系统
  - 底盘系统
  - 车身外观
  - 电气系统
  - 制动系统
  - 转向系统
  - 传动系统
  - 空调系统

### 搜索配件
- 支持按配件名称搜索
- 支持按配件编码搜索
- 支持按配件分类筛选
- 搜索结果支持分页显示

## 客户管理操作

### 新增客户
1. 点击「客户管理」
2. 点击「新增客户」
3. 填写客户信息：
   - 客户名称
   - 联系人
   - 联系电话
   - 收货地址
4. 点击「保存」

### 客户查询
- 支持按客户名称搜索
- 支持按联系人搜索
- 点击客户行可查看详细信息

## 数据统计

### 可查询的统计信息
- 本月入库总数量
- 本月销售出库最多的配件
- 当前库存总数量
- 配件种类数量
- 配件分类数量
- 待审批出库单数量

### 统计查询方式
- 在 AI 助手中直接提问
- 例如："本月入库多少件"
- 例如："哪种配件出库最多"
- 例如："当前库存总量是多少"
