## 💡 接下来该做什么？

### 方案一：安装Maven（推荐用于完整运行）

#### 选项1：手动下载安装（最稳妥）

1. 访问 https://maven.apache.org/download.cgi
2. 下载 `apache-maven-3.9.6-bin.zip`（或最新版本）
3. 解压到 `C:\Program Files\Apache\maven`
4. 配置环境变量：
   - 新增系统变量 `MAVEN_HOME` = `C:\Program Files\Apache\maven`
   - 在Path中添加 `%MAVEN_HOME%\bin`
5. 验证：`mvn -version`

详细步骤请查看：[环境安装指南.md](环境安装指南.md)

#### 选项2：先安装Chocolatey再安装Maven

如果你想要更简单的包管理方式：

1. 以管理员身份运行PowerShell，执行：
```powershell
Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))
```

2. 然后安装Maven：
```powershell
choco install maven -y
```

## 📝 项目快速启动流程（完整版本）

安装完Maven后：

1. **配置MySQL到PATH**（可选但推荐）
   - 参考 [配置MySQL环境.bat](配置MySQL环境.bat)

2. **创建数据库**
   - 使用Navicat或命令行执行 `database/init.sql`

3. **配置后端数据库密码**
   - 修改 `application.yml` 中的数据库密码

4. **启动后端**
   ```powershell
   cd auto-parts-warehouse-backend
   mvn spring-boot:run
   ```

5. **启动前端**
   ```powershell
   cd auto-parts-warehouse-frontend
   npm run serve
   ```

6. **访问系统**
   - 前端：http://localhost:8081
   - 后端API：http://localhost:8080
   - 默认账户：admin / admin123
